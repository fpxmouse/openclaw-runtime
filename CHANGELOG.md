# Changelog

## v1.0.0

- Add Debian-based runtime layer for OpenClaw images.
- Add Python tooling: pip, venv, build tools, uv, and common libraries.
- Add Node/npm support for JavaScript-based skills.
- Add Docker CLI for hosts that mount `/var/run/docker.sock`.
- Add automatic skill dependency scanning for `setup.py`, `pyproject.toml`, `requirements.txt`, and `package.json`.
- Add `ocr doctor`, `ocr repair`, and `ocr install-skills`.
- Add GHCR build workflows and Unraid documentation.
